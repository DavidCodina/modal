Modal Demo (No bootstrap.js)

Overview:

This demo uses the basic bootstrap.css modal styles, but with a few tweaks. Note that if you were not going to include bootstrap.css, you'd want to grab the modal styles and integrate them with the following code.


  .modal {
    display: block;
    visibility: hidden;
    background-color: transparent;
  }

  .modal-open { max-height: 100vh; }

  .modal.show { visibility: visible; }

From here, we add the following styles:


  .modal.animate-open {
    -webkit-animation-name:     darken-background;
    animation:                  darken-background;
    -webkit-animation-duration:  1.5s;
            animation-duration:  1.5s;
    -webkit-animation-fill-mode: forwards;
            animation-fill-mode: forwards;
  }

  .modal.animate-open .modal-content {
    -webkit-animation-name:     backInDown;
    animation:                  backInDown;
    -webkit-animation-duration: 1.5s;
    animation-duration:         1.5s;
  }

  .modal.animate-close {
    -webkit-animation-name:     lighten-background;
    animation:                  lighten-background;
    -webkit-animation-duration:  1.5s;
            animation-duration:  1.5s;
    -webkit-animation-fill-mode: forwards;
            animation-fill-mode: forwards;
  }

  .modal.animate-close .modal-content {
    -webkit-animation-name:     backOutUp;
    animation:                  backOutUp;
    -webkit-animation-duration: 1.5s;
    animation-duration:         1.5s;
    -webkit-animation-fill-mode: forwards;
            animation-fill-mode: forwards;
  }
